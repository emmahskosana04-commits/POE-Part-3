# POE Part 3 - Glow Beauty Salon

## 1. Project Overview
Final implementation of Glow Beauty Salon website for POE Part 3. Website for a beauty team in Bethal Mpumalanga offering Hair, Nails, Braids, Cuts, and Skincare services.

## 2. Refinements from Part 2 Feedback
Feedback, Problem, Solution Implemented

-Nav not responsive, Duplicate ☰ button showed on desktop, Added `@media (min-width: 769px)` to hide ☰ on desktop 
-Links not visible, Mobile nav links faint on background, Changed to `color: black` + `font-weight: 700` + white dropdown 
-No dynamic content, Static copyright year, Added JS `new Date().getFullYear()` + live date/time with `setInterval()` 

## 3. Part 3 New Features
1. **JavaScript**: Live date + time in footer updates every second. Auto copyright year.
2. **Responsiveness**: Mobile 384px, Tablet 768px, Desktop 1920px using media queries
3. **SEO**: Meta description + keywords in `<head>`. Alt text on all images.
4. **Going Live**: Files structured for FTP upload to hosting server.

## 4. Technologies Used
- HTML5, CSS3 Flexbox + Media Queries, Vanilla JavaScript
- VS Code for development
- GitHub for version control

## 5. How to View
1. Clone: `git clone https://github.com/emmahkosana04-commits/POE-Part-3`
2. Open `index.html` in browser
3. Test mobile: Press F12 → Device toolbar → Set width to 384px
